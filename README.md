# BIENVENIDO/A AL PROYECTO DE MAINSAIT, INDRA
------------------------------------
1º DAM, curso 2024-2025. Daniel Lleó
------------------------------------
### Este repositorio de GitHub contiene todos los directorios y cuestiones del proyecto demandadas por la empresa.
Un cliente ha solicitado la creación de una página web sobre conferencias, talleres y eventos sobre la sostenibilidad.
Para ello, se han de trabajar diferentes cuestiones del desarrollo de software: programación, bases de datos, lenguaje de marcas, etc.
------------------------------------
### A continuación, una pequeña guía de la estructura del proyecto:
- /doc: en este directorio encontrarás todas las cuestiones relacionadas con la documentación del proyecto, incluyendo pruebas del Windows 10 instalado en la máquina virtual.
- /html: en este directorio encontrarás todo el contenido relacionado con la página web, incluyendo HTML, CSS e imágenes usadas en la página.
- /sql: en este directorio encontrarás todos los scripts de SQL demandados para el proyecto.
- /src: en este directorio encontrarás una carpeta "main" con el código fuente en C# de las diferentes clases necesarias para el proyecto y una carpeta "test" con diferentes pruebas unitarias para el proyecto.
------------------------------------
### Puede que este README se vaya actualizando conforme se avance en el proyecto.
